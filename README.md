# Yandex Cloud Terraform Example
Проект создания облачной инфраструктуры разработки для платформы Kubernetes в Яндекс Облаке.
Для автоматизации применяется Terraform.

Структура:
* doc/ - документация
* terraform/ - скрипты автоматизации
